NutriFact

Nutrifact is an app that takes an image of the barcode of most grociery sold products, and automatically tracks your intake, as well as warns you about allergies, etc.

OneArmBand

OneArmBand is the combination of several technologies including the LinkIt ONE and the Myo Band. The project was 1st place winner at the AT&T's 2015 Boston Hackathon.

MakeToMove

This project was created for Fab11 Boston, and was presented on Sunday August 9th, 2015. Controlled by two arduinos, the project was to create a giant geodesic dome which incouraged people to move!

AutoHotKey Master

AutoHotKey Master is simply the autohotkey file I used in order to create a huge amount of shotcuts from my computer, primarily using the ` key.

VimDocs

VimDocs is a google docs addon which adds vim keybindings.

ShortcutFinder

ShortcutFinder is a Java GUI program which reads txt files to create a dynamic search list for shortcuts.

BattleshipChess

BattleshipChess is a combination of Battleship and Chess, which integrates battleship like gamestyle, but chess style turns. Created using Unity.

IN6GC

IN6GC stands for Ipod Nano 6th Generation Calender. It uses the google calender api in order to create some images which can be synced to the ipod to see the events for the day.
